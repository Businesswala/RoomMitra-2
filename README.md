# RoomMitra | Rental & Living Marketplace

RoomMitra is a modern, mobile-first rental and living marketplace designed specifically for student and co-living ecosystems. It connects tenants, roommates, and service providers (Tiffin & Laundry) directly.

## 🚀 Key Modules

*   **Authentication & KYC**: Secured via email/password, social Google Login, mobile OTP verification, and Aadhaar/PAN identity audits.
*   **Property Listings**: Rooms, hostels, and PGs with advanced geolocation filtering (city, budget, amenities) and views analytics.
*   **Ecosystem Services**: Mapped tiffin meal package builders and laundry pickup/delivery details grids.
*   **Roommate Matching**: Algorithmic compatibility calculations matching roommate profiles.
*   **Real-time Messenger**: Private chat streams, typing receipts, and image attachments.
*   **SaaS Subscriptions**: silver, Gold, and Premium plans integrated with Razorpay/PhonePe.
*   **Admin KPI Panels**: Moderation audits, user account controls, CMS pages, and verification reviews.

## 🛠️ Technology Stack

*   **Backend**: Laravel 12 (PHP 8.3), MySQL 8+, Redis Cache & Queue, Laravel Sanctum, Laravel Reverb (WebSockets).
*   **Mobile client**: Flutter SDK (Riverpod, Dio, GoRouter, Secure Storage, Firebase messaging).
*   **Website**: Laravel Blade templates, Tailwind CSS, Alpine.js.

## 📂 Project Structure

*   `laravel-backend/`: Laravel REST API, database seeds, and Blade views.
*   `flutter-frontend/`: Dart Flutter client codebase.
*   `docs/`: Configuration handbooks, API specs, audits, and guides.
